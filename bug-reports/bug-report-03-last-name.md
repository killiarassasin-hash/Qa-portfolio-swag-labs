## ID
BUG-03

## Заголовок: 
Покупка товара с отсутствующим Last Name
## Окружение:

-Приложение:Swag Labs

-Тип приложения:Web

-Браузер:Chrome

-OC:Windows
## Предусловия:

-Пользователь авторизован под логином error_user
-В корзине находятся товары
## Шаги воспроизведения:

1. Ввести логин без пароля

2. Нажать Enter

3. Ввести First Name

4. Ввести Last Name

5. Ввести Zip/Postal Code
## Ожидаемый результат: 
Товар оформляется, пользователя переносит на страницу подтверждения оплаты
## Фактический результат:
Поле Last Name заблокировано, система оформляет заказ, перенося пользователя в окно подтверждения оплаты, но не давая её совершить 
## Cерьёзность:
Critical
## Приоритет:
High
## Примечания: 
Нарушена бизнес-логика интернет-магазина.
